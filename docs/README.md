## Tech Stack

- **Frontend**: Next.js 15 (React, App Router)
- **Backend**: Next.js API Routes
- **Database**: Prisma + Postgres
- **Deployment**: Vercel
- **Blockchain**: Base Sepolia & Mezo Testnet
- **Smart Contracts**: Solidity + OpenZeppelin
